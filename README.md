# EE656 Course Project
## Self-Optimal Clustering using Threshold Function

This repository contains the implementation and analysis of our course project for **EE656: AI, Machine Learning, Deep Learning and its Applications** at **Indian Institute of Technology Kanpur**.

### Project Overview

The project implements **Self-Optimal Clustering (SOC)**, an enhanced version of the **Improved Mountain Clustering (IMC)** algorithm. The objective is to automatically determine optimal cluster centers by incorporating a threshold function that improves cluster compactness while maintaining better separation between clusters.

The project includes:
- Implementation of the SOC algorithm
- Comparison with the baseline IMC method
- Experimental evaluation on benchmark datasets
- Performance analysis using clustering metrics
- Visualization of clustering results

---

## Features

- Implementation from scratch
- Automatic determination of cluster centers
- Threshold-based optimization
- Evaluation using standard clustering metrics
- Visual comparison of clustering performance
- Experimental analysis on multiple datasets

---

## Evaluation Metrics

The implementation evaluates clustering quality using:

- Global Silhouette Index (GSI)
- Partition Index (PI)
- Separation Index (SI)
- Cluster Compactness
- Cluster Separation
